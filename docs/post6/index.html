Complementario I - Molde

Creación de Molde

Para crear un molde y poder copiarlas, es necesario fabricar huellas podotáctiles para tener los moldes y poder vaciar la resina con el objetivo de crear un modelo. Luego de vertir la resina líquida, que está creada a partir de la mezcla de dos líquidos los cuales tendrán una reacción química y con el tiempo se solidificarán. Ahí es cuando es posible desmoldar y tener múltiples copias de la misma pieza.

Pasos - Creación de Molde en Fusion360

Paso a paso para crear un molde con el software Fusion 360.

1. Líneas Iniciales

Hacer un ‘sketch’ y crear una línea en el eje frontal de forma vertical para crear una línea de referencia. Luego otra linea generando la forma de la imágen.

2. Revolucionar la Geometría.

Generar un ‘Revolve’ teniendo como perfil la gometría creada anteriormente y como eje la línea de referencia creada al principio.

Lo cual, generará el cuerpo a continuación.

3. Fillet

Hacer un ‘Fillet’ en las dos curvas que se ven en la imágen.

4. Crear Base para Molde

Desde la cara inferior y apagando el sólido, crear un rectángulo que comprenda la figura anterior.

Esta se debe extruir, para crear el molde. Este debe ser mayor que la el primer cuerpo creado.

5. Cambiar Material

Para poder diferenciar ambos cuerpos, se le dará un nuevo material a la huella.

6. Combinar

Con la huella y el molde creado, seleccionaremos la herramienta ‘Combine’, en la cual definiremos a la molde como el ‘target body’ y la huella como ‘tool body. La función ‘Cut’ debe estar seleccionada y debo apretar la opción ‘Keep Tools’.

Con esto, se verá como la huella quedó marcada en el molde.

7. Análisis de Sección

En ‘Inspect’, debemos seleccionar ‘Section Analisis’ para ver cómo será el molde por dentro.

8. Ángulo de Desmolde

Para que el molde tenga un ángulo de desmolde, primero debemos crear una nueva línea de referencia de forma vertical desde el centro.

Crear una nueva geométría que genere un ángulo de desmolde.

A esta, nuevamente hacer un revolve con la geometría de ‘Profile’ y la línea de referencia como ‘Axis’

Así, esa cara se le restará al molde y por lo tanto el ángulo de desmolde será constante en todo el molde.
